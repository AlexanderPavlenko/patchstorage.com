Loopy Record on Note Received

loopy

This is a helper script for Loopy. It is like the “Wait for Note’ that some MIDI recorders have that start recording when the first note is received.

For this to work, you will need to set up a MIDI Binding in Loopy for the “Toggle Record and Select Next Track” action. You will also need to route MIDI from your MIDI controller to this script.

Tap the “Wait to Record On/Off” to turn it on (it lights up when it is on). When the next MIDI note comes in, it sends out the MIDI command to Loopy that will start it recording.
